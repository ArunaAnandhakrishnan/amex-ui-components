# Release Notes

A full changelog of all component library versions, including new components, improvements, and breaking changes.

---

## v1.2.0 — *Current*
**Released:** March 2026

### ✨ New Components
- **FileUpload** — Drag-and-drop or click-to-upload with multi-file support
- **NotificationToast** — Auto-dismissing feedback toasts with 4 variants
- **DateInput** — Native date picker with min/max range and error state
- **SearchBar** — Pill-shaped search input with clear button and CVA support

### 🔧 Improvements
- `Modal` — Added `closeOnBackdrop` input and `hasFooter` slot support
- `Card` — Added `hoverable` lift animation and `hasFooter` slot
- `Accordion` — Added `multiple` mode to allow multiple open panels simultaneously
- `Tabs` — Added `disabled` support per tab item
- All CVA components now correctly propagate `setDisabledState`

### 🐛 Bug Fixes
- `Select` — Fixed arrow icon overlapping long option text
- `ProgressBar` — Fixed indeterminate animation clipping on Safari
- `Checkbox` — Fixed checkmark alignment on Windows Chrome

---

## v1.1.0
**Released:** February 2026

### ✨ New Components
- **Badge** — Status indicator with 6 variants and 3 sizes
- **Avatar** — User representation with initials or image fallback
- **Spinner** — Animated SVG loading indicator
- **Toggle** — On/off switch with CVA support
- **Tag** — Removable label with 6 color variants
- **Divider** — Horizontal/vertical separator with optional label
- **ProgressBar** — Determinate and indeterminate progress indicator
- **Alert** — Contextual feedback with dismissible support
- **IconButton** — Circular icon-only button (primary, ghost, danger)
- **Breadcrumb** — Navigation trail with configurable separator
- **Pagination** — Smart page navigator with ellipsis for large ranges
- **Card** — Content container with image, header and footer slots
- **Modal** — Overlay dialog with sm/md/lg sizes
- **Tabs** — Switchable content panels
- **Accordion** — Collapsible FAQ-style sections

### 🔧 Improvements
- Storybook upgraded to v10 with Vite builder — faster HMR
- All components switched to inline templates for Vite compatibility
- Added `@storybook/addon-essentials` — Actions and Controls tabs now fully functional
- `FormGroup` — Added `horizontal` layout mode

---

## v1.0.0
**Released:** January 2026

### 🎉 Initial Release

#### Atoms
| Component | Selector | Description |
|-----------|----------|-------------|
| Button | `ui-button` | Primary action trigger with variant and size |
| Input | `ui-input` | Text, email, password, number input with CVA |
| Textarea | `ui-textarea` | Multi-line text input with resize |
| Select | `ui-select` | Custom-styled native dropdown |
| Checkbox | `ui-checkbox` | Binary selection with custom styling |
| RadioGroup | `ui-radio-group` | Single-select from a list of options |

#### Molecules
| Component | Selector | Description |
|-----------|----------|-------------|
| FormField | `ui-form-field` | Label + hint + error wrapper |
| FormGroup | `ui-form-group` | Fieldset grouping related fields |

### 🏗 Infrastructure
- Angular 21 + Nx 22 monorepo setup
- `libs/ui` — component library with Storybook
- `libs/ui-styles` — SCSS design tokens (colors, spacing, typography)
- Design tokens: `$color-primary`, `$color-secondary`, `$color-text`, `$color-border`, `$color-success`, `$color-error`, `$color-warning`
- Spacing scale: `$spacing-xs` (4px) → `$spacing-xl` (32px)
- Typography: `$font-size-sm/md/lg`, `$font-weight-normal/bold`

---

## Versioning Policy

This library follows [Semantic Versioning](https://semver.org/):

- **Patch** `x.x.Z` — Bug fixes, no API changes
- **Minor** `x.Y.0` — New components or backwards-compatible features
- **Major** `X.0.0` — Breaking changes to component APIs
